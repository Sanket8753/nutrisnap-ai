# NutriSnap AI 🍏📸

NutriSnap AI is a full-stack web application designed to help users seamlessly track their nutrition and weight goals. By leveraging advanced AI vision models, users can simply upload screenshots or photos of their meals to receive instant estimations of calories, protein, and overall breakdown. 

## 🚀 Features

- **AI Food Photo Analysis:** Upload meal photos/screenshots to instantly detect food items and estimate macronutrients (Calories, Protein).
- **Quantity Adjuster:** Fine-tune meal portion sizes and weights dynamically after AI identification.
- **Weight Log & Progress Tracker:** Log daily weight changes.
- **Data Visualizations:** Beautiful weekly and monthly progress graphs for calorie metrics and weight trends.

## 🛠️ Tech Stack

- **Frontend:** React (Vite), Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ODM)
- **AI Engine:** Google Gemini API
